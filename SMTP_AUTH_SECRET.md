# QQ 邮箱授权码存储

此处仅用于本地记录您的 QQ 邮箱授权码，请勿提交到代码仓库（Git）。

## 您的授权码

```text
在此处粘贴您的授权码
```

## 如何获取

1. 登录 QQ 邮箱网页版。
2. 进入 **设置** -> **账号**。
3. 下滑找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**。
4. 开启 **POP3/SMTP服务**。
5. 点击 **生成授权码**。

## 使用说明

请在 **营销系统 -> 设置 -> 发件邮箱管理** 中，编辑您的默认发件账户，将此授权码填入 **SMTP 密码/授权码** 字段。
